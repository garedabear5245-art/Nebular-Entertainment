How to Open HTML Files Offline

  1. Go to your Files app.
  2. Find and open the HTML file.
  3. If it says "This file type is not allowed", continue below.
  4. Open your browser History.
  5. Find the blocked file → 3-dot menu → More from this site.
  6. Add ?id=800 and change start to file:///
